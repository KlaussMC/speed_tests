Note:
There are some resources provided to run sample programs, 
the resources that are not available will be pre-compiled. (located in the Compiled folder)

If you wish to Recompile these files, Visual Studio / JDK with the necessary tools is required.

To Run PHP, Place the contents of the PHP_RESOURCES folder into another folder named PHP
and add it to the PATH Environment variable. then type "PHP" followed by the filename.
